# Solitario – Proyecto de Python

## Descripción:
Juego de cartas tipo Solitario desarrollado desde cero en Python. En colaboración de Santino Meoqui. 
Este proyecto implementa la lógica del juego con clases modulares (`Baraja`, `Carta`, `Mano`, `Solitario`) y se ejecuta desde la terminal.

El objetivo principal es demostrar buenas prácticas de programación orientada a objetos y diseño modular de un proyecto funcional.

## Tecnologías:
- Python 3
- Programación orientada a objetos
- Uso de clases, métodos, algoritmos y estructuras básicas

## Estructura del proyecto
- Solitario-Proyecto-Python-/
  - core/
    - main.py             # Archivo principal para ejecutar el juego
    - Baraja_inglesa.py   # Clase Baraja
    - Clase_Carta.py      # Clase Carta
    - Clase_Mano.py       # Clase Mano
    - Logica_Solitario.py # Clase Solitario y lógica del juego
  - README.md

## Cómo ejecutar
1. Clonar el repositorio:
   git clone https://github.com/TU_USUARIO/Solitario-Proyecto-Python-.git
2. Ir a la carpeta core:
   cd Solitario-Proyecto-Python-/core
3. Ejecutar el juego:
   python main.py

## Cómo jugar
- El juego se ejecuta en la terminal.
- Sigue las instrucciones en pantalla para mover cartas y jugar Solitario.


Notas: 
- Se evitó el uso de clases externas al máximo posible, el proyecto funciona de forma autónoma y solo requiere de la clase random integrada en python.
- Se reconoce la falta de snake_case y otros estándares en el código; el proyecto se realizo con caracter educativo y no se considero el uso de estos inicialmente.
- A partir de este proyecto aprendimos la utilidad de buenas prácticas al codificar.
